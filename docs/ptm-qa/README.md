# ptm-qa · 质量工程师

> 状态：⬜ 未开始 · 所属项目：ptm-team · Step 3 规划中

## 角色定位

**质量策划与度量工程师**。质量数据的采集者和分析者，不直接参与测试执行，但为整个团队提供质量可视化。

## 核心职责

| 职责 | 说明 |
|---|---|
| 质量策划 | 制定质量标准和度量体系 |
| 过程审计 | 审计用例评审/测试过程/交付物质量 |
| 数据采集 | 采集缺陷密度/覆盖率/通过率等质量指标 |
| 趋势分析 | 生成质量趋势报告和风险预警 |
| 质量报告 | 输出质量状态评估给 ptm-tm |

## 工作流程

```
ptm-qa
  ├── 接收 ptm-tm 下发的质量检查任务
  ├── 执行过程审计（用例评审/测试过程/交付物）
  ├── 质量数据采集（缺陷密度/覆盖率/通过率）
  └── 输出 → 质量状态评估 → ptm-tm 汇总报告
```
