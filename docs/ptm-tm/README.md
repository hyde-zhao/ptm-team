# ptm-tm · 测试经理

> 状态：⬜ 未开始 · 所属项目：ptm-team · Step 1 规划中

## 角色定位

**测试经理 / 对外协调责任人**。团队与外部世界的接口——把版本目标拉入团队，把风险状态推出去。

## 核心职责

| 职责 | 说明 |
|---|---|
| 项目计划 | 接收项目/版本转测通知，制定整体测试计划（范围/策略/时间/资源） |
| 调度协调 | 调度下游 Agent（tse/tde/te/qa）完成各阶段任务 |
| 进度跟踪 | 跟踪执行进度，识别风险，向上汇报 |
| 对外协调 | 与禅道、钉钉、干系人对接 |

## 关键调用链

```
ptm-tm
  ├── 调用 notification-skill（公共）推送风险预警
  ├── 调用禅道集成工具 跟踪任务状态
  └── 输出 → 项目执行摘要 → ptm-tse 策略调整
```
